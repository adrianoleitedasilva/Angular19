Para instalarmos o Angular, vamos precisar acessar o site angular.dev/installation

```cmd
npm install -g @angular/cli@19.0.2
```

**Verificando a versão da instalação**

```
ng --version
```

**Criando nosso primeiro projeto**
```
ng new nomeDoProjeto
```

```
ng new conceitos-basicos
```

![[Pasted image 20260302155450.png]]

Para executar o projeto e ver o que temos até o momento, vamos no terminal e acessar a pasta de conceitos-basicos:

```
cd conceitos-basicos
```

E em seguida, vamos rodar o comando:

```
ng serve
```

![[Pasted image 20260302160238.png]]

Para acessar o localhost, vamos no navegador e digitar:
```
http://localhost:4200
```

Se tudo der certo vamos ver a seguinte página:

![[Pasted image 20260302160413.png]]

Esse é o template inicial que vemos na versão 19.0.2 do Angular.
