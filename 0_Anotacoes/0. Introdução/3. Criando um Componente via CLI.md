Agora vamos criar um componente usando a Command Line Interface usando o comando. 
```prompt
ng generate component minhapagina
```

Ao executarmos esse comando veremos as seguintes mensagens no terminal:
![[Pasted image 20260302183202.png]]

E ao olharmos a pasta app, vamos ver que uma nova pasta foi criada com o nome que colocamos e são criados 4 arquivos.
![[Pasted image 20260302183133.png]]

Olhando os arquivos criados, temos:

```html

E para ver esse component, precisamos ir nos arquivos ***app.component.ts*** e no ***app.component.html*** e fazer os **imports**

```ts
import { Component } from '@angular/core';
import { RouterOutlet } from '@angular/router';
import { HelloWorldComponent } from './helloworld/hello.component';
import { MinhapaginaComponent } from './minhapagina/minhapagina.component';

@Component({
  selector: 'app-root',
  imports: [RouterOutlet, HelloWorldComponent, MinhapaginaComponent],
  templateUrl: './app.component.html',
  styleUrl: './app.component.scss',
})

export class AppComponent {
  title = 'conceitos-basicos';
}
